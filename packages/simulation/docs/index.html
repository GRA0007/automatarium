closureWithPredicate

  • closureWithPredicate(graph: FSAGraph, currentStateID: number, predicate: ClosureWithPredicateFn): Set<[number, Transition[]]>
    • Defined in closureWithPredicate.ts:26

    Compute the set of states accessible from a given state using only transitions that meet the conditions as defined by the given predicate.

    example

    Find states accessible using only transitions that read 'A'.

    const closure = closureWithPredicate(graph, 0, transition => transition.read.includes('A'))
    const states = Array.from(closure).map(([state, path]) => state)
    example

    Compute lambda closure from a given state.

    closureWithPredicate(graph, 0, transition => transition.read.length === 0)

    Parameters

    • graph: FSAGraph

      Graph object used as input

    • currentStateID: number

      ID of state used to seed the closure

    • predicate: ClosureWithPredicateFn

      Predicate function

    Returns Set<[number, Transition[]]>

    A set of pairs where each contains an accessible state and the path of transitions to it

expandReadSymbols

  • expandReadSymbols(read: string): string[]
    • Defined in expandReadSymbols.ts:51

    Expands and de-dupes the symbols represented by a read string.

    example

    Expand a string representing a run of lowercase letters

    const symbols = expandReadSymbols('[a-e]')
    console.log(symbols) // ['a', 'b', 'c', 'd', 'e']
    example

    Expand and de-dupe a string representing specific letters and a run of digits

    const symbols = expandReadSymbols('abb[3-5]')
    console.log(symbols) // ['a', 'b', '3', '4', '5']

    Parameters

    • read: string

      the string to expand

    Returns string[]

    The set of unique symbols represented by the read string
